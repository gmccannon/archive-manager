# Start up guide (to run locally)

Install docker  
Start the docker daemon  
In the root directory run:  
```docker compose up --build```  
Navigate to [localhost 3000](http://localhost:3000/)  

## Tech Stack  
Node.js, React, Flask, Docker, Vercel, Google Cloud Run
